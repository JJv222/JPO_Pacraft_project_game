# JPO_Pacraft_project_game
Simple original Pacman game, in style of Minecraft.    
Project made by SFML library in C++. 

~Project made for my studies, 4th semester. <br />
You can choose one of 3 difficulties level: <br />
-Easy <br />
-Normal <br />
-Hard <br />
